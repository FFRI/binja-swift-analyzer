# SwiftMetadataDump

A tool to extract Swift metadata using [go-macho](https://github.com/blacktop/go-macho) and display it in JSON format.

## Usage

```
go build
go install
SwiftMetadataDump <path to file> <metadata to be parsed> <architecture>
```
